Summary

The exercise consists in correcting the script.js file in the assignment 4 solution starter folder, that loops over an array of names and prints out either a hello or goodbye to the browser console. If the name starts with a letter j or J, it prints out Goodbye JSomeName. If the name starts with any other letter, it prints out Hello SomeName.
